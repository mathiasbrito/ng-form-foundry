Field layout — appearance.grid / minFieldWidth

grid lays a node's scalar fields on a CSS grid (cols per row, left-to-right; rows alone fills top-to-bottom, adding columns as needed) and overrides minFieldWidth, which instead fits as many equal-width fields per row as stay at least that wide. Without either, fields share one wrapping flex row, where minTextFieldWidth / minNumberFieldWidth bound how narrow string and number fields may get. The appearance is set on the root group only — the nested TLS group and the Backend choice inherit it (a child setting its own layout would override).
