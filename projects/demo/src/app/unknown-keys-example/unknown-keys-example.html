Partial-schema editing — unknownKeys

A hand-written, srsRAN-flavored libconfig file is edited through a JSON Schema that covers only pci, dl_arfcn, and an optional log_level the file doesn't have yet. The unknownKeys option decides what happens to everything else: preserve keeps uncovered settings verbatim on save, drop deletes them, and edit surfaces them as editable fields typed by the document's own literals. In every mode, a value the file writes in hex displays and edits as hex — the document, not the schema, decides the base.
